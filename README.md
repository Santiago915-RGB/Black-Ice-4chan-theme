## Lengthy CSS document for theming 4chan

### Style motives
- Goth
- Mac OS 9
- Cool colors

### Fonts used
- freeSerif
- Monaco 9
- Mona12
- TX-02 (for banner image in README.md)

### Installation
Just copy the `css` file to your clipboard and go to any 4chan board, see the top right corner, go to settings, scroll to the bottom until you find the Custom CSS label, check it and click the braced `[edit]` text and paste the css
